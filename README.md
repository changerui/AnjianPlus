## 這是什麼

按鍵精靈 Plus 是一套以**原生 C++ 從零重寫**的桌面自動化工具，保留你熟悉的 Q 語言與操作邏輯，同時解決原版在現代環境的種種痛點：

- **全程 UTF-8**：告別原版 Big5/GBK 中文亂碼，匯入舊腳本時自動轉換編碼。
- **輕量免安裝**：主程式 ~5 MB、播放器 ~2.6 MB，解壓即用，不動系統登錄機碼。
- **相容原生 .Q 腳本**：直接開啟或匯入按鍵精靈 2014 的腳本，語法相容、語意一致。
- **原生插件免註冊自動載入**：把插件 `.dll`/`.ocx` 丟進 `plugin` 資料夾，命令自動出現；支援大漠 dm、Chrome 等常見 COM 插件。

---

## 主要功能

### IDE（TaskRunner.exe）

- **語法高亮 + 自動完成**：命令、關鍵字、變數即時提示。
- **智能糾錯**：偵測拼錯的命令名稱，一鍵套用建議修正。
- **逐步執行 + 參數監測**：一行一行執行，即時顯示每個變數的值，不靠猜就能除錯。
- **界面設計器**：拖拉設計腳本 UI 視窗（UIPackage 格式，與原版相容）。
- **取色器**：從螢幕任意位置取色，直接插入腳本。
- **記憶體掃描器**：CheatEngine 風格的記憶體搜尋與數值凍結，掛載於工具選單。
- **命令庫 + 函式說明**：內建完整命令參考，支援以 `.qml` 擴充命令庫。
- **腳本庫管理**：依分類管理腳本，免手動開資料夾。
- **更新提醒**：啟動時檢查 GitHub 是否有新版，有更新時主畫面顯示提示橫幅，一鍵前往下載頁取得新版（正式版 / 測試版雙通道）。

### 播放器（player.exe）

- **自包含執行**：單一 exe，引擎已 bytecode 保護，可獨立發布給使用者執行腳本。
- 支援所有 IDE 可用的 Q 語言指令、COM 插件、UIPackage 界面匯入。
- 無需安裝 IDE 即可執行已完成的腳本。

## 安裝與使用

**系統需求：Windows 10 或更新版本（64 位元）**

1. 到 [Releases](../../releases) 下載最新版 `TaskRunner-x.x.x.zip`。
2. 解壓到任意資料夾（建議獨立資料夾，例如 `C:\TaskRunner\`）。
3. 雙擊 `TaskRunner.exe` 即可啟動。

**新增插件：**
把插件的 `.dll`（或 `.ocx`）及同名 `.ini` 放入程式目錄的 `plugin\` 資料夾，重新啟動程式後命令自動出現。

**新增命令庫：**
把 `.qml` 檔放入 `lib\` 資料夾，重新啟動後可在命令清單看到。

> **防毒誤報說明**：自動化工具因操作鍵盤滑鼠、讀寫記憶體，常被防毒軟體誤報。這是業界普遍現象，程式本身無惡意。如遇誤報，請將程式資料夾加入信任清單，或向防毒廠商提交誤報申訴。

## 更新提醒

程式啟動時會自動檢查 GitHub Releases 是否有新版本。有更新時主畫面頂部會出現提示橫幅，點擊即可前往下載頁取得新版。你的腳本與設定都存放在程式目錄，下載新版覆蓋程式檔即可，資料不受影響。

也可在「程式設定 → 程式更新」手動檢查，或切換正式版 / 測試版通道。
