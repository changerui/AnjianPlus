# AnjianPlus · 按鍵精靈 Plus

> 從零打造、全程 UTF-8、64 位元的新一代桌面自動化工具。
> 讓電腦自動幫你完成重複的辦公、報表與網頁行政工作。

**AnjianPlus** 是一套**自有原始碼**、功能對等並超越經典按鍵精靈的自動化軟體。它完整支援按鍵精靈的 **Q 語言腳本**、**原生 `.Q` 腳本匯入**與**原生插件（dll/ocx）免註冊載入**，並以現代化的開發體驗重新打造編輯器與執行引擎。發行版以綠色免安裝的形式提供，丟進資料夾即可使用。

## ✨ 主要功能

- **Q 語言引擎**：變數/陣列、流程控制、Sub/Function、鍵鼠操作、找圖找色、視窗、記憶體讀寫、檔案、字串與數學等完整命令庫。
- **原生相容**：可直接匯入按鍵精靈的原生 `.Q` 腳本（自動處理 Big5/GBK 編碼）與界面封包；支援第三方 COM 插件（如 chrome、大漠 dm 等）**免註冊**自動載入——丟進 `plugin` 資料夾命令就自動出現。
- **現代化編輯器**：語法高亮、自動完成、智能糾錯（拼錯命令一鍵修正）、自動縮排與區塊補完、整理排版。
- **逐步執行 + 參數監測**：一行一行跑、即時看每個變數的值，方便除錯。
- **內建腳本庫**：依分類管理你的腳本，免開系統檔案總管。
- **雲端同步**：用你自己的 GitHub repo 當雲端，腳本一鍵推送 / 拉取。
- **自動更新**：內建線上更新（正式版 / 測試版雙通道），保留你的腳本與設定。

## 🚀 下載與使用

1. 到 [Releases](../../releases) 下載最新版的 `TaskRunner-x.y.z.zip`。
2. 解壓到任意資料夾（綠色免安裝）。
3. 執行 `TaskRunner.exe` 即可開始。
4. 想擴充插件：把插件的 `.dll/.ocx`（+ 可選的 `.ini`）丟進程式目錄的 `plugin` 資料夾，命令會自動出現。

---

## 🔄 自動更新

程式內建線上更新：開啟後會自動檢查新版本，有更新時主畫面會提示，一鍵即可下載換檔並重啟，過程**保留你的腳本與設定**。也可在「⚙ 程式設定 → 程式更新」手動檢查，並切換**正式版 / 測試版**通道。
