## 這是什麼

按鍵精靈 Plus 是一套**自有原始碼**、全程 UTF-8、64 位元的桌面自動化工具。它：

- 保留你熟悉的 **Q 語言**與操作邏輯，老腳本、舊習慣不用重學。
- 能直接**匯入按鍵精靈的原生 `.Q` 腳本**，自動處理 Big5/GBK 編碼，**告別亂碼**。
- 支援**原生插件（`.dll/.ocx`）免註冊自動載入**——丟進 `plugin` 資料夾，命令就自動出現（連大漠 dm、chrome 這類也行）。
- **綠色免安裝**，解壓即用，不動你的系統。

---

## ✨ 主要功能

- **現代化編輯器**：語法高亮、自動完成、**智能糾錯**（拼錯命令一鍵修正）、自動縮排與區塊補完、一鍵整理排版。
- **逐步執行 + 參數監測**：一行一行跑，即時看每個變數的值，除錯不再靠猜。
- **完整命令庫**：鍵鼠操作、找圖找色、視窗、記憶體讀寫、檔案、字串、數學、流程控制、UI 對話框……該有的都在。
- **內建腳本庫**：依分類管理你的腳本，不用再開檔案總管東翻西找。
- **雲端同步**：用你自己的 GitHub 當雲端，腳本一鍵推送 / 拉取，換台電腦也帶著走。
- **自動更新**：內建線上更新（正式版 / 測試版雙通道），更新時**保留你的腳本與設定**。

## 🚀 下載與使用

1. 到 [Releases](../../releases) 下載最新版的壓縮檔。
2. 解壓到任意資料夾（綠色免安裝）。
3. 執行裡面的主程式即可開始。
4. 想擴充插件：把插件的 `.dll/.ocx`（＋可選的 `.ini`）丟進程式目錄的 `plugin` 資料夾，命令會自動出現。

> 小提醒：自動化工具常被防毒軟體**誤報**，這是業界普遍現象、不是程式有問題。如遇誤報，可將程式加入信任清單，或依正規管道向防毒廠商提交誤報申訴。

---

## 🔄 自動更新

開啟程式後會自動檢查新版本，有更新時主畫面會跳出提示，一鍵即可下載換檔並重啟，過程**完整保留你的腳本與設定**。也可在「⚙ 程式設定 → 程式更新」手動檢查，並切換**正式版 / 測試版**通道。
